# Research

My research sits at the intersection of **LLM-powered autonomous agents** and **Financial AI (AI4Finance)**. I build systems that bridge foundation model capabilities with real-world decision-making — from general computer control to quantitative trading.

---

## Research Interests

- **LLM Agents & Multi-Agent Systems** — Hierarchical agent frameworks, tool use, long-horizon planning, and agent orchestration protocols (e.g., [AgentOrchestra](https://arxiv.org/abs/2506.12508), [Cradle](https://arxiv.org/abs/2403.03186))
- **AI4Finance** — End-to-end financial platforms, algorithmic trading, portfolio management, and high-frequency trading via RL and LLMs (e.g., [FinWorld](https://dl.acm.org/doi/proceedings/10.1145/3711896), [TradeMaster](https://proceedings.neurips.cc/paper_files/paper/2023))
- **Reinforcement Learning** — Deep RL for sequential decision-making in complex, partially observable environments
- **Embodied AI** — Aligning LLMs with interactive environments through experience and practice

---

## Selected Projects

### AgentOrchestra
A hierarchical multi-agent framework built around the **Tool-Environment-Agent (TEA) protocol**. AgentOrchestra enables flexible orchestration of specialist sub-agents for complex, long-horizon tasks. It achieves strong performance on general web and computer control benchmarks.
[[Paper]](https://arxiv.org/abs/2506.12508)

---

### Cradle
A foundation agent framework for **general computer control** — enabling agents to operate arbitrary desktop software using only screen pixels and natural language instructions, without task-specific APIs. Cradle demonstrates the potential of vision-language models as general-purpose computer users.
[[Paper]](https://arxiv.org/abs/2403.03186)

---

### FinWorld
An **all-in-one open-source platform** for end-to-end financial AI research and deployment. FinWorld integrates data pipelines, model training, backtesting, and live deployment into a unified framework, lowering the barrier for rigorous AI4Finance research.
*KDD 2026*
[[Paper]](https://dl.acm.org/doi/proceedings/10.1145/3711896)

---

### TradeMaster
A holistic **quantitative trading platform** powered by reinforcement learning, covering data processing, environment simulation, agent training, and performance evaluation across multiple financial markets and trading tasks.
*NeurIPS 2023*
[[Paper]](https://proceedings.neurips.cc/paper_files/paper/2023)

---

### EarnHFT
An **efficient hierarchical RL framework for high-frequency trading**, decomposing the trading problem into macro-level strategy selection and micro-level order execution for improved sample efficiency and profitability.
*AAAI 2024*
[[Paper]](https://ojs.aaai.org/index.php/AAAI/article/view/29413)

---

### Multimodal Foundation Agent for Financial Trading
A **tool-augmented, diversified, and generalist** multimodal agent for financial trading that integrates heterogeneous data sources and diverse trading tools, achieving state-of-the-art results across multiple financial benchmarks.
*KDD 2024*
[[Paper]](https://dl.acm.org/doi/proceedings/10.1145/3637528)
